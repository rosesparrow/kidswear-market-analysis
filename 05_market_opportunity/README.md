# 05_market_opportunity

This folder quantifies the **market opportunity** for a premium teen (11–19) loungewear/pyjamas brand in the UK.

## Notebooks

- **`market_gap_analysis.ipynb`**  
  Core gap thesis + evidence synthesis from prior folders (competitors, search trends, social, strategic models).  
  Includes rough SAM sizing using demographic forecasts.

- **`market_demographics.ipynb`**  
  Historical UK population trends (2011–2024) + cohort-component forecast (2025–2035) for ages 5–19 and core 11–19.  
  Shows demographic stability/replenishment despite low fertility — key support for addressable market size.

## Key Outputs
- Evidence that the £55–85 premium teen segment is largely uncontested (evidence from competitor pricing and positioning maps).
- Forecasted 11–19 core audience ~7.5M in 2025 (base case), stable with migration buffer.
- Rough girls-focused SAM estimate: £7.8M–£26.1M annual potential (conservative 3–10% penetration).
